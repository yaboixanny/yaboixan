---
description: Automatically sync local changes to GitHub repository
---

This workflow will add, commit, and push any local changes to the main branch on GitHub.

1. Stage all changes
// turbo
2. git add .

3. Create a commit with a descriptive message
// turbo
4. git commit -m "auto: update from antigravity session"

5. Push to the main branch
// turbo
6. git push origin main
